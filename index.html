Project 5

Binary Search and Jump Search are efficient algorithms used to find elements in a sorted array or list. Binary Search works by repeatedly dividing the array into two halves and checking the middle element to determine where the target value may be located. On the other hand, Jump Search works by jumping ahead by fixed steps to find the block where the element might exist. After locating the block, it performs a linear search within that section to find the exact element.
